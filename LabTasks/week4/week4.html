Task 1: Generating a jSymbolic

In generating a jSymbolic of my piece, I have selected a group of features that I believe can be interesting for my piece:

  • Number of Pitches: 35
  • Number of Pitch Classes: 10
  • Number of Common Pitches: 2
  • Number of Common Pitch Classes: 1
  • Range: 53
  • Strong Tonal Centres: 2
  • Mean Pitch: 57.66
  • Mean Pitch Class: 5.521
  • Most Common Pitch: 57
  • Most Common Pitch Class: 9
  • Interval Between Most Prevalent Pitches: 7
  • Pitch Variablity: 9.46
  • Most Common Melodic Interval: 0
  • Mean Melodic Interval: 4.576
  • Number of Common Melodic Intervals: 6
  • Distance Between Most Prevalent Melodic Intervals: 2
  • Repeated Notes: 0.2041
  • Mean Tempo: 91.88
  • Tempo Variability: 14.77
  • Minimum Note Duration: 0.0631
  • Maximum Note Duration: 3.506
  • Average Note Duration: 0.3784
  • Voice Equality-Range: 2.828
  • Variation of Dynamics: 6.662

I have extracted some basic features like the number of individual pitches and pitch classes present in a song, mean pitch and pitch class, range and so on, but I have also decided to extract some features which I believe can provide useful insight into this specific song. Specifically, for example, I have chosen to extract Mean Tempo and Tempo Variability since there are 3 changes of tempo within my piece. Further, I have decided to extract data about note duration as this piece is consisted of a lot of very short notes. Also, there are multiple dynamics changes in the song, so I have extracted data about variation of dynamics.

Task 2: Generating a piano roll and a pitch histogram

Pitch histogram

The extracted pitch histogram shows the number of times a particular note has been played in a song. According to this histogram, the most common pitch in my piece is A3, which is consistent with the data extracted from jSymbolic as MIDI pitch value of 57 is equivalent to A3.
Piano Roll

The extracted piano roll, like any piano roll, shows the order, length, and pitch of the notes played accross time. It seems to be consistent with the rest of analytical data from this page (as seen when comparing it with the historgram for example, since the commonality of notes presented in the histogram seems to correspond to the piano roll).
Scatter Plot of Pitches

The extracted scatter plot of pitches plots pitch against note length. Again, this seems to be consistent with my song as, for example, according to the graph the semibreve notes present in a song are A and C# which is consistent with the actual song.
