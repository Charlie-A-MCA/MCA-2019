Week 10: Audio Similarity and Transcription

Task 1: Generating a Similarity Matrix

For this part of the task, I have used the same tracks from the previous 2 weeks. In the similarity matrix, tracks 0-3 fall within the classical genre, tracks 4-6 fall within the rock genre, and tracks 7-9 are my tracks. Track 7 is Mary Roose, and is classified as blues, track 8 is Gradual Sunrise and falls within rock, while track 9 is Lullaby and is classified as classical. The matrix reflects this as tracks 0-3 and track 9 are more similar to each other than any other tracks. Tracks 4-8 are also similar to each other, although there is not much difference between rock songs and the blues song. However, this can be explained with the fact that rock and blues are generally quite similar genres.

Task 2: Transcription

For this task, I have decided to transcribe the file I generated in week 2, that is, my Little Black Submarines file. This was done by first exporting the MuseScore file into a wav file, then importing it into Sonic Visualiser and transforming it into a piano roll which was exported as a MIDI file. The MIDI file was then opened by MuseScore and exported as an image. Below is a page-by-page comparison of the original notated file I created (on the left) and the transcribed file from this week (on the right):

The difference between the files is apparent. The first difference is that the automatically transcribed file has the entire song written in two staves, while the original file has a voice staff and and 2 staves for the piano. Secondly, the original file has a quadruple (4/4) time signature, while the transcribed file has a triple (3/4) time signature. Thirdly, tempo of the transcribed file is different than that of the original file. The list could go on, as there is not a single bar that looks the same in both files.
When it comes to how both files sound, expectedly they sound quite different. The part of the transcribed file which sounds the most similar to the original file in my opinion corresponds to the bars 74 to around 100 in the transcribed file.
